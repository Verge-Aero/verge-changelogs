# AERO Studio - 2026.3.4.0

## Features

- [**External Files**] Added designer hot reload for external file references, allowing supported media changes on disk to refresh without reloading the full show
- [**External Files**] Added a File References panel showing referenced files, owners, status, and contextual actions
- [**External Files**] Added scene-level ownership tracking for external file references, including scene objects, components, events, and graph nodes
- [**External Files**] Added targeted refresh paths for image planes, projector image/video events, audio events, texture file sources, and formation graph texture nodes
- [**File References Panel**] Provides file reference owner display with nested entries, tooltips, hover highlighting, double-click focus, and right-click actions
- [**Scene Hierarchy**] Added search support for scene objects and component types
- [**Scene Hierarchy**] Clearing search keeps selected objects visible by expanding their hierarchy paths
