# AERO Studio - 2026.3.3.0

## Features

- [**Editor Viewport**] Added a button that toggles a "split" view, showing the render and editor view simultaneously
- [**Timeline**] Added button to select all events nested in the selected scene objects

## Improvements

- [**Point Cloud Paint**] Gradients can now be painted where the falloff is the interpolation between ends of the gradient
- [**Point Cloud**] Added undo/redo support for point cloud editing
- [**Timeline**] Added panning with middle-click for vertical and horizontal movement anywhere in the timeline panel
- [**Formation Graph**] Added option to choose the origin for image-to-formation nodes

## Changes

- [**Formation Graph**] Defaults image-to-formation node to resolve to center

## Bug Fixes

- [**Scene View**] Fixed a single drone always rendering at the scene origin
